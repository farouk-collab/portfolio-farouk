# Portfolio Farouk

Portfolio personnel realise avec React, Vite, Tailwind CSS v4 et Framer Motion.

## Scripts

- `npm run dev` lance le serveur de developpement
- `npm run build` genere la version de production
- `npm run preview` previsualise le build localement

## Deploiement Render

Le projet inclut un fichier `render.yaml` pour un deploiement en site statique.

Configuration principale :

- Build command : `npm install && npm run build`
- Publish directory : `dist`

## Contenu

- Hero premium avec palette vert emeraude
- Presentation du profil DevOps / Cloud
- Projets techniques
- Competences structurees
- Consultation et telechargement du CV
- Bloc contact avec liens directs
